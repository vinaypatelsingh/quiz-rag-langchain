SECTION: JavaScript Basics
--------------------------------------------------

Q1: What is JavaScript?
A: JavaScript is a high-level, interpreted programming language used to create interactive web applications.

Q2: Difference between let, const, and var?
A:
- var: function scoped, can be redeclared
- let: block scoped, cannot be redeclared
- const: block scoped, cannot be reassigned

Q3: What is a closure?
A: A closure is a function that remembers variables from its lexical scope even when executed outside that scope.

Q4: What is the event loop?
A: The event loop handles asynchronous operations in JavaScript by managing the call stack and callback queue.

SECTION: Practical Coding Questions

Q28: Reverse a string in JavaScript.
A:
function reverseString(str) {
  return str.split('').reverse().join('');
}

Q29: Check if a number is prime.
A:
function isPrime(num) {
  if (num <= 1) return false;
  for (let i = 2; i < num; i++) {
    if (num % i === 0) return false;
  }
  return true;
}

Q30: Find the largest number in an array.
A:
function max(arr) {
  return Math.max(...arr);
}
